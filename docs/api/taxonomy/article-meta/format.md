# Format

Format metadata (`meta_type=format`) describes **how a story is structured**, not what it is about.

## Filter

```text
?meta=format:news_story
```

Discover values in your project with `format_categories` from [Article facets](../../articles/facets.md) or `GET …/articles/metadata/types/format/values`.
