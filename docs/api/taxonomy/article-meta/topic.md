# Topic

Topic metadata (`meta_type=topic`) describes **broad subject areas** — politics, education, health, sports, and so on. Articles may carry multiple topic tags.

## Filters

```text
?meta=topic:local_government_politics
```

Discover values in your project with `topic_categories` from [Article facets](../../articles/facets.md) or `GET …/articles/metadata/types/topic/values`.
