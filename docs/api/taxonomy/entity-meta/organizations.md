# Organizations

`organization_type` classifies canonical organizations, organization mentions, and organization search filters.

Discover values in your project with `GET …/organizations/types` — see [Organizations types](../../organizations/types.md).

## Categories

| Value | Typical use |
| --- | --- |
| `government` | General government body |
| `law_enforcement` | Police department or similar |
| `court` | Court or judicial body |
| `legislative_body` | Legislature or council |
| `political_party` | Political party |
| `school_district` | School district |
| `school` | K–12 school |
| `university` | College or university |
| `hospital` | Hospital or health system |
| `public_health` | Public health agency |
| `public_services` | Other public service agency |
| `utilities` | Utility provider |
| `company` | General company |
| `local_business` | Local business |
| `financial_institution` | Bank or financial firm |
| `real_estate` | Real estate firm or developer |
| `nonprofit` | Nonprofit organization |
| `community_group` | Community or civic group |
| `religious_org` | Religious institution |
| `culture_arts` | Museum, theater, or arts org |
| `sports_team` | Sports team |
| `sports_league` | League or athletic association |
| `media` | News outlet or media company |
| `other` | Does not fit another category |
