# People

`person_type` classifies canonical people, person mentions, and people search filters.

Discover values in your project with `GET …/people/types` — see [People types](../../people/types.md).

## Categories

| Value | Typical use |
| --- | --- |
| `athlete` | Professional or amateur athlete |
| `coach` | Sports coach |
| `sports_official` | Referee, umpire, or league official |
| `sports_executive` | Team or league executive |
| `elected_official` | Elected officeholder |
| `government_official` | Appointed or career government official |
| `political_staff` | Staff to elected or political officials |
| `lawyer_legal_advocate` | Attorney or legal advocate |
| `judge_court_official` | Judge or court officer |
| `law_enforcement_public_safety` | Police, fire, EMS, or similar |
| `crime_justice_subject` | Person central to a crime or justice story |
| `business_owner_executive` | Business owner or executive |
| `business_professional` | Other business professional |
| `labor_union_representative` | Union leader or representative |
| `artist_entertainer` | Performer or entertainer |
| `media_journalism` | Journalist or media figure |
| `arts_culture_professional` | Curator, director, or cultural professional |
| `education_research_expert` | Educator or researcher |
| `healthcare_worker` | Clinician or health worker |
| `community_member` | Resident or community participant |
| `unknown` | Type could not be determined |
| `other` | Does not fit another category |

Legacy extractor values such as `politician` normalize to `elected_official`.
