# Locations

Location mention `nature` values describe the editorial role of a place **in one story**. Used when filtering location mentions, geo search, geo-cells, and location hub routes.

## Categories

| `nature` | Meaning |
| --- | --- |
| `primary` | Main geographic focus of the story |
| `secondary` | Relevant but not the main place |
| `subject` | Place is the central subject of the narrative |
| `context` | Background or setting |
| `person` | Place tied to a person's biography (lived, worked, from) |
| `unknown` | Role could not be classified |
| `other` | Other role |

Example filters: `?nature=primary` on [Geographic search](../../articles/geo-search.md) or [List locations hub](../../articles/hub/locations.md).
