# Organizations

Organization mention `nature` values describe the editorial role of an organization **in one story**. Used on organization hub routes, organization entity article/mention lists, and mention search with `entity_type=organization`.

## Categories

| `nature` | Meaning |
| --- | --- |
| `primary` | Main organization in the story |
| `actor` | Organization taking action or making decisions |
| `source` | Organization as information source |
| `subject` | Organization is the central subject |
| `affected` | Organization affected by events |
| `regulator` | Regulator or oversight body |
| `context` | Background organization |
| `other` | Other role |

Example: `?nature=actor` on [List organizations hub](../../articles/hub/organizations.md).
