# People

Person mention `nature` values describe the editorial role of a person **in one story**. Used on person hub routes, person entity article/mention lists, and mention search with `entity_type=person`.

## Categories


| `nature`      | Meaning                                    |
| ------------- | ------------------------------------------ |
| `subject`     | Person is the central subject of the story |
| `source`      | Person quoted or providing information     |
| `expert`      | Subject-matter expert                      |
| `official`    | Official acting in an official capacity    |
| `witness`     | Witness to events                          |
| `affected`    | Person affected by events                  |
| `victim`      | Victim of crime or harm                    |
| `suspect`     | Person suspected or accused                |
| `participant` | Participant in events                      |
| `observer`    | Observer or bystander                      |
| `context`     | Background figure                          |
| `other`       | Other role                                 |


Example: `?entity_type=person&nature=subject` on [List mentions hub](../../articles/hub/mentions.md).
