# Enrichment nodes

**Enrichment nodes** refine or look up the details an [extractor](extractors.md) found, adding information that wasn't in the original text.

| Node | What it adds |
| --- | --- |
| **Geocode** | Turns a place name into map coordinates and a normalized address, so locations can be plotted and searched geographically |

Geocoding can also draw on your [Stylebook](../../stylebook/index.md) catalog to recognize places you've already mapped, which improves consistency and reduces repeated lookups.
