# Support

Stuck on something? Here's how to get help.

!!! note "Stub"
    Support channels will be finalized closer to launch.

## Channels

- **FAQ** — check the [frequently asked questions](faq.md) first.
- **Email** — reach the team at [support@backfield.dev](mailto:support@backfield.dev).
- **Issues** — bug reports and feature requests (tracker link coming soon).

## Reporting a bug

When reporting an issue, include:

1. What you were trying to do
2. What happened instead, including any error messages
3. Request IDs from API responses, if applicable
