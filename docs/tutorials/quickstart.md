# Quickstart

This tutorial walks through creating your first Backfield project, connecting a data source, and triggering a pipeline run.

!!! note "Stub"
    Steps below are placeholders until the platform is available.

## 1. Install the CLI

```bash
pip install backfield
```

## 2. Authenticate

```bash
backfield auth login
```

## 3. Create a project

```bash
backfield init my-first-project
cd my-first-project
```

## 4. Run a pipeline

```bash
backfield run
```
